AngelAid

About Project

AngelAid is a mobile app that promotes the idea of the "Kampung Spirit". Users can use the app to request for help from nearby neighbours such as watering of plants when you are going overseas or simply just request for any cooking ingrediants that you might be lacking if anybody is willing to help you with that. An additional app promote website is also done to promote the app and is fully coded using html, css and javascript. This project is a group project and my role of the project is the main developer/coder for the mobile app.

Software Used

  • Adobe XD
  • HTML5
  • CSS
  • Javascript
  • Brackets
